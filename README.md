#  Hospital Management System

A web-based Hospital Management System developed using PHP and MySQL for managing hospital operations efficiently. The system allows administrators to manage patients, doctors, appointments, departments, staff, prescriptions, lab reports, and medical records.



##  Features

* Patient management
* Doctor management
* Department management
* Appointment scheduling
* Staff management
* Room management
* Prescription management
* Lab report management
* Surgery management
* Medical records management
* Database integration and CRUD operations



##  Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* SQL
* XAMPP/WAMP Server
* CRUD Operations

##  Installation and Setup

1. Install XAMPP.
2. Move the project to `htdocs`.
3. Start Apache and MySQL.
4. Create and import the database using phpMyAdmin.
5. Open:


http://localhost/Hospital%20Management%20System/
